# Icon Placeholder untuk PWA Testing
# Ukuran: 512x512 pixels
# Warna: #0b1326 (background), putih (text)

Untuk membuat icon nyata:
1. Gunakan tool online seperti https://favicon.io
2. Upload logo atau desain sederhana
3. Generate semua ukuran
4. Letakkan di folder public/

Atau gunakan command berikut jika memiliki ImageMagick:
```
magick convert -size 512x512 xc:#0b1326 -fill white -font Arial -pointsize 200 -gravity center -draw "text 0,0 '💰'" icon-512x512.png
```
